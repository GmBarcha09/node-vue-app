# test-task

## Setup - Frontend

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start

```

## Setup - Backend

```bash
# install dependencies
$ yarn install

# To run the migrations
$ yarn db migrate:latest

# serve with hot reload at localhost:4000
$ yarn dev

# build for production

$ yarn build


```
